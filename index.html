Privacy Policy for Tab Manager

Tab Manager does not collect, store, or transmit any personal data.

All data (including tab information, pinned groups, themes, and saved URLs) is stored locally in the user's browser using Chrome storage APIs.

We do not:

  • Track users
  • Use analytics
  • Serve ads
  • Share data with third parties

Permissions usage:

  • tabs: used to display and organize open tabs
  • storage: used to save user preferences locally
